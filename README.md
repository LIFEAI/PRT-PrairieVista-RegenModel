# PRT — Prairie Vista Estates Regenerative Model

**Town of Vulcan, Alberta · 63 Acres · ASP Bylaw 1358-08**

An interactive diagnostic and regenerative redesign of Prairie Vista Estates, built on the Planetary Regenerative Trust (PRT) framework.

## What's Inside

- **Diagnosis** — 8 fatal flaws of the 2008 ASP
- **New Model** — 5 blended neighborhood clusters + Village Heart
- **Land Use** — Regenerative vs. conventional land use comparison
- **Five Capitals** — Scoring: ASP 22/100 → Regenerative 79/100
- **PRT Capital** — Lane A/B/C investment architecture + RCCS credits
- **Phasing** — 4-phase implementation with readiness gates
- **Next Steps** — 7 priority actions with contacts and checklist

## Framework

- Planetary Regenerative Trust (PRT)
- Regenerative Capital Credit System (RCCS v2.0)
- Local Regenerative Land Trust (LRLT)
- Story of Place (Bill Reed / Regenesis Group)
- Five Capitals Framework
